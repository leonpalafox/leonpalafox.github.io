# leonpalafox.com
Leon Palafox's personal webpage.

## SEO and Domain Notes

- Canonical domain in site metadata is `https://www.leonpalafox.com`.
- Keep `CNAME` set to `www.leonpalafox.com`.
- In GitHub Pages settings, keep `www.leonpalafox.com` as the primary custom domain and enable HTTPS enforcement so apex/http traffic is redirected to HTTPS on the primary host.
